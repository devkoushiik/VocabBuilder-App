# Dark Theme Improvements Applied

## Changes Made:

1. **Softer Dark Theme Color Scheme** - 3-color combination:
   - Soft Dark Gray (#1a1d29, #252836, #2d3142)
   - Soft Purple accents (#a78bfa, #8b5cf6, #c4b5fd)
   - Soft Teal accents (#14b8a6, #5eead4)

2. **Gradient Borders** - Added to cards in dark mode using:
   - Purple → Teal → Purple gradient
   - Applied via GradientBorder component

3. **Filter Fonts** - Changed to semibold (fontWeight: '600')

4. **Text Contrast** - Fixed throughout:
   - White backgrounds use black text
   - Dark backgrounds use white/light text
   - All text elements now use theme colors for proper contrast

The theme is now much softer and easier on the eyes while maintaining excellent readability!
